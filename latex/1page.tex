\section{Implementation}
For the implementation part of our project, we have decided to make a quote generator using RNNs. For this implementation, we used the Pytorch library because we found it more intuitive and easier to debug then other frameworks. First, we build our dictionary by extracting all the different words from the dataset. We then create an embedding layer which is responsible to learn a vector representation for each of our words. We have integrated the possibility to initialize our embedding layer with a pretrained GloVe model if we which to. The network is composed of an embedding layer which is responsible to learn a vector representation of our bag of words. We trained our network using a quote dataset of size 36.2k that we found on Kaggle \cite{quote}.
